# GUI Kullanım Kılavuzu

## Başlatma

GUI'yi başlatmak için:

```bash
cd gui
python3 main.py
```

veya proje kök dizininden:

```bash
python3 gui/main.py
```

## Özellikler

- **Tümünü Başlat**: Server ve Client 2'yi otomatik başlatır
- **Tümünü Durdur**: Tüm servisleri durdurur
- **Veri Gönder**: Metin girişi ve hata tespit yöntemi seçimi
- **Sonuçlar**: Tüm işlemlerin logları ve sonuçları görüntülenir

## Kullanım

1. "Tümünü Başlat" butonuna tıklayın
2. Gönderilecek metni girin
3. Hata tespit yöntemini seçin
4. "Gönder" butonuna tıklayın
5. Sonuçları log alanında görüntüleyin
